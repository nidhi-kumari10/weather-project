# 🌦️ Weather App

A simple and elegant weather application that allows users to check real-time weather conditions of any city. Built using **HTML, CSS, and JavaScript**, this app provides key weather details like temperature, humidity, and wind speed with a clean UI.

🌐 **Live Demo:** https://nidhi-kumari10.github.io/weather-project/

---

## ✨ Features

* 🔍 Search weather by city name
* 🌡️ Display current temperature
* 💧 Shows humidity levels
* 🌬️ Displays wind speed
* 🎨 Modern gradient UI design
* ⚡ Fast and responsive interface

---

## 🚀 Tech Stack

* **HTML5** – Structure of the app
* **CSS3** – Styling and layout (gradient UI)
* **JavaScript** – Logic and API handling

---

## ⚙️ How It Works

1. Enter a city name in the search bar
2. Click the search icon 🔍
3. The app fetches weather data from an API
4. Displays:

   * Temperature 🌡️
   * City Name 📍
   * Humidity 💧
   * Wind Speed 🌬️

## 🛠️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/nidhi-kumari10/weather-project.git
```

2. Navigate to the project folder:

```bash
cd weather-project
```

3. Open `index.html` in your browser
